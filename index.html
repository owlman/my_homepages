欢迎光临

owlman.org 是一个 sitemap 性质的、极简易的个人主页。目的是以外链的形式汇聚本人日常使用的博客、Github 以及出版书籍的信息，以供交流之用。本站遵循极简主义的风格，本着够用即可的原则，使用纯 HTML5 + CSS3，采用响应式布局。开发过程中不使用任何第三方库、框架或者 CSS 主题，在编辑器中用纯代码的形式编写而成。并采用 Github Pages 的形式发布（源码在此）。
